# Aether AI Agency Website

Official website for **Aether AI** — a premium AI agency building autonomous agents and intelligent systems.

Minimal, dark, and architectural by design, the site emphasizes invisible intelligence, subtle motion, and clarity over hype.

---

## Live Site

https://aether-01.web.app

---

## Stack

- HTML5  
- Tailwind CSS (CDN)  
- Vanilla JavaScript  
- Canvas-based particle system  
- Inter and Geist fonts  

---

## Features

- Responsive single-page layout  
- Ambient particle and glass effects  
- Smooth scrolling and subtle animations  
- Lightweight, framework-free implementation  

---

## Usage

Open directly in a browser or deploy on any static host such as Vercel, Netlify, or Cloudflare Pages.
